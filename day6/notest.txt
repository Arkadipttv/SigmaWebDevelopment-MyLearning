Core Web Vitals

CLS - Cumulative Layout Shift
page load hote hote kitna shift ho raha hai

CLS is a measure of the largest burst of layout shift scores for every unexpected layout shift that occurs during the enitre lifespan of a page.

LCP
largest Contentful paint

A good lcp score is that the page should load within 2.5 seconds or less.
reports the render time of the largest image or text block visible within the viewport, relative to when the page first started loading.

FID
First INput Delay

IF a user clicks on a button in a webpage, the time taken for the page to respond. This delay in the time is measured using FID
A good FID is 100 milliseconds or less
